GitHub 接続設定

▲

GitHubと連携することで、このページからイベントを編集し、公開サイトに反映できます。

1 GitHubリポジトリのURLを貼り付ける
例: https://github.com/yamada-taro/antigravity-events
2 接続情報を確認する（自動入力済み）
3 アクセストークンを入力する
トークンはこのブラウザにのみ保存されます。他の人と共有しないでください。

アクセストークンの取得方法

GitHubのアクセストークンはイベントデータの更新に必要な「鍵」です。以下の手順で取得してください。

  1. github.com にアクセスし、管理者アカウントでログインしてください。
  2. 右上のプロフィール画像をクリックして、Settings（設定） を選択してください。
  3. 左側のメニューを一番下までスクロールし、Developer settings をクリックしてください。
  4. Personal access tokens → Tokens (classic) の順にクリックしてください。
  5. Generate new token → Generate new token (classic) をクリックしてください。
  6. Note（メモ） に AG Admin と入力し、有効期限（Expiration）を選択してください。
  7. スコープ（権限）の一覧から repo にチェックを入れてください。
  8. ページ下部の Generate token ボタンをクリックしてください。
  9. ghp_ で始まるトークンが表示されます。コピーして設定欄のアクセストークン欄に貼り付けてください。
    このトークンは一度しか表示されません。必ずコピーしてください。
